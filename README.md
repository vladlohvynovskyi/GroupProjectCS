#How to run the game:

1. Create the virtual environment
python3 -m venv .venv

2. Activate the virtual environment

Linux/macOS :
source .venv/bin/activate

Windows PowerShell :
.\venv\Scripts\Activate.ps1

3. Install dependencies
pip install -r requirements.txt

4. Run the game
python3 main.py
